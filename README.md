- [Disclaimer](#disclaimer)

# Get Up And Running
* Install dependencies:
  ```sh
  # Using yarn
  yarn

  # Using npm
  npm install
  ```
* Copy and rename `env.example` to `env` and update its values.
* Start the app: `yarn dev`
